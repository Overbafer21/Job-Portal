# 🚀 Карьерный Портал (Job Portal System)

Небольшое настольное Java Swing-приложение для учета заявок на вакансии.

## Что реализовано

- Полностью русскоязычный интерфейс.
- Удобная форма ввода и таблица заявок.
- Добавление новой заявки с базовой валидацией полей.
- Очистка формы одной кнопкой.
- Таблица с начальными тестовыми записями.

## Стек

- Java 8+
- Swing (`javax.swing`)

## Запуск

```bash
javac JobPortalSystem.java
java JobPortalSystem
```

## Структура

- `JobPortalSystem.java` — основной код приложения и UI.

## Возможные улучшения

- Валидация email по регулярному выражению.
- Поиск и фильтрация по должности.
- Сохранение/загрузка заявок в файл (JSON/CSV).
- Редактирование и удаление существующих заявок.
